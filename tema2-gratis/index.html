2.1 El escritorio de Windows
Barra de tareas, menú Inicio, iconos y accesos directos
El escritorio es la pantalla principal de Windows. Contiene iconos, accesos directos y la barra de tareas.
La barra de tareas se sitúa por defecto en la parte inferior. Incluye el botón Inicio, zona de búsqueda, aplicaciones ancladas y la bandeja del sistema (reloj, iconos de notificación, red, volumen).
El botón Inicio (tecla Windows ⊞) abre el menú Inicio con las aplicaciones instaladas, acceso a Configuración, documentos recientes y apagado.
Los accesos directos son enlaces a programas o archivos. Se crean con botón derecho → Crear acceso directo. Tienen una pequeña flecha en el icono.
Escritorios virtuales (Windows 10/11): permiten organizar el trabajo en múltiples escritorios. Se gestionan con el botón Vista de tareas o con Win + Tab. Añadir nuevo escritorio: Win + Ctrl + D.
La barra de búsqueda de Cortana (Windows 10) permite buscar archivos, aplicaciones y en la web directamente desde el escritorio.
En Windows 11, el menú Inicio está centrado y rediseñado. Los widgets aparecen en la barra de tareas y ofrecen información en tiempo real.
📁
2.2 Explorador de archivos
Navegación, operaciones con archivos y carpetas
El Explorador de archivos (Win + E) permite navegar por la estructura de carpetas del sistema. El panel izquierdo muestra el árbol de directorios.
Operaciones básicas: copiar (Ctrl+C), cortar (Ctrl+X), pegar (Ctrl+V), eliminar (Supr), renombrar (F2), seleccionar todo (Ctrl+A).
La Papelera de Reciclaje almacena temporalmente los archivos eliminados. Para eliminar definitivamente: Mayús + Supr. Para vaciar: botón derecho → Vaciar Papelera.
Propiedades de archivo: botón derecho → Propiedades. Muestra tamaño, ubicación, fecha de creación, atributos (solo lectura, oculto).
Búsqueda de archivos: barra de búsqueda en el Explorador (parte superior derecha) o Win + F. Admite filtros por fecha, tipo, tamaño.
Vistas disponibles: iconos grandes/medianos/pequeños, lista, detalles (recomendada), mosaicos, contenido.
Rutas de carpetas importantes: C:\Users\[usuario] (perfil de usuario), C:\Windows (sistema), C:\Program Files (programas de 64 bits), C:\Program Files (x86) (programas de 32 bits).
⚙️
2.3 Panel de control y Configuración
Gestión del sistema, hardware y software
El Panel de control (clásico) permite ajustar configuraciones avanzadas del sistema. Se accede desde el menú Inicio → buscar "Panel de control".
La app Configuración (Win + I) es la interfaz moderna. Incluye: Sistema, Dispositivos, Red e Internet, Personalización, Aplicaciones, Cuentas, Hora e idioma, Accesibilidad, Seguridad.
Programas y características (Panel de control): permite instalar y desinstalar programas, ver actualizaciones instaladas, activar/desactivar características de Windows.
Administrador de dispositivos: gestiona el hardware del equipo. Permite actualizar, deshabilitar o desinstalar controladores (drivers). Acceso: Win + X → Administrador de dispositivos.
Opciones de energía: define planes de energía (equilibrado, ahorro de energía, alto rendimiento). Configura qué ocurre al cerrar la tapa o pulsar el botón de encendido.
El Área de notificaciones (Centro de actividades en Win 10, Win 11 Panel de acceso rápido): notificaciones del sistema y accesos rápidos (WiFi, Bluetooth, modo avión, brillo).

2.4 Cuentas de usuario y seguridad
Tipos de cuenta, permisos y Control de cuentas de usuario
Tipos de cuenta en Windows: Administrador (acceso total al sistema) y Usuario estándar (acceso limitado, no puede instalar programas ni cambiar configuración crítica).
Microsoft Account: cuenta en línea que sincroniza configuración, archivos OneDrive y preferencias entre dispositivos. Alternativa: cuenta local (solo para ese equipo).
El Control de cuentas de usuario (UAC) solicita confirmación o credenciales de administrador cuando una acción puede afectar al sistema. Protege contra cambios no autorizados.
Contraseña y PIN: el PIN de Windows Hello es más rápido que la contraseña y solo funciona en ese dispositivo. También admite huella dactilar y reconocimiento facial.
Opciones de inicio de sesión: Configuración → Cuentas → Opciones de inicio de sesión. Permite configurar la pantalla de bloqueo, inicio automático y tiempo de espera.
♿
2.5 Accesibilidad
Narrador, lupa, teclado en pantalla y más
Las herramientas de accesibilidad en Windows se encuentran en Configuración → Accesibilidad (Win 11) o Configuración → Accesibilidad/Facilidad de acceso (Win 10).
El Narrador lee en voz alta el contenido de la pantalla para personas con discapacidad visual. Se activa con Win + Ctrl + Enter.
La Lupa (Win + +) amplía parte o toda la pantalla. Útil para personas con baja visión. Se cierra con Win + Esc.
El Teclado en pantalla permite escribir sin teclado físico. Útil para usuarios con movilidad reducida. Se activa desde Accesibilidad o buscando "teclado en pantalla".
El Alto contraste cambia los colores de la pantalla para mejorar la visibilidad. Atajo: Alt + Mayús izq + Impr Pant.
Texto de cursor: aumenta el grosor del cursor de texto para mejor visibilidad. Puntero del mouse: permite cambiar tamaño y color del cursor.
🛡️
2.6 Mantenimiento y seguridad básica
Windows Update, Defender, copias de seguridad
Windows Update: sistema de actualizaciones automáticas. Las actualizaciones de seguridad se instalan automáticamente. Se gestiona en Configuración → Windows Update.
Windows Defender (ahora Seguridad de Windows): antivirus y antimalware integrado en Windows. Incluye protección en tiempo real, firewall y control de aplicaciones.
Firewall de Windows: bloquea conexiones entrantes no autorizadas. Se configura en el Panel de control → Sistema y seguridad → Firewall de Windows Defender.
Historial de archivos: copia de seguridad automática de documentos, imágenes y música en un disco externo. Configuración → Actualización y seguridad → Copia de seguridad.
Restaurar sistema: permite volver a un punto de restauración anterior si el sistema falla. Panel de control → Sistema → Protección del sistema → Restaurar sistema.
Administrador de tareas (Ctrl + Mayús + Esc o Ctrl + Alt + Supr → Administrador de tareas): muestra procesos activos, rendimiento de CPU/RAM/disco, servicios y aplicaciones de inicio.
Liberador de espacio en disco: elimina archivos temporales, caché, miniaturas y archivos de la papelera para liberar espacio. Buscar "Liberador de espacio en disco".
